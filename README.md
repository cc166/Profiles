# Profiles

这个仓库用于存放我自己的配置模板，当前优先整理：

- OpenClash 配置模版
- Loon 配置模版
- tvOS 的 Loon 配置模版

## 模板目录

- `templates/openclash/openclash-template.yaml`
- `templates/loon/loon-template.conf`
- `templates/loon-tvos/loon-tvos-template.conf`

## 规则来源

这些模板默认示例接入我自己的规则合集仓库：
- `https://github.com/cc166/ShuntRules`
- 合集入口：`https://github.com/cc166/ShuntRules/blob/main/docs/CUSTOM-COLLECTIONS.md`

当前可直接接入的合集包括：
- AI
- Communication
- Development
- Media
- DomesticBasic
- Platform
